# NJU 2024级 MBA 选课助手

一个基于金币竞价系统的MBA选课辅助工具，帮助学生规划选课策略。

## 在线访问

访问地址：https://miobowl.github.io/nju24MBAcpicker/

## 功能特点

- 🎯 **智能金币分配**：按优先级权重自动分配1000枚金币
- 📅 **日历视图**：直观展示3-8月课程安排
- ⚠️ **冲突检测**：自动识别时间冲突的课程
- 💾 **本地存储**：选课数据自动保存到浏览器
- 📱 **响应式设计**：支持桌面和移动设备

## 使用方法

1. 直接在浏览器中打开 `index.html`
2. 将课程从左侧列表拖拽到中间的优先级栏（必选/想选/可选）
3. 系统自动计算建议金币分配
4. 查看日历视图确认课程时间安排
5. 检查并处理可能的时间冲突

## 技术栈

- 纯 HTML5 + CSS3 + Vanilla JavaScript
- 原生 Web APIs：Drag and Drop、LocalStorage、CSS Grid
- 零依赖，无需构建

## 文件结构

```
├── index.html          # 主应用（单文件）
├── courses.json        # 课程数据库
├── course_data.json    # 备用课程数据
└── docs/               # 文档目录
```

## 金币分配算法

- **必选课程**：权重 × 3
- **想选课程**：权重 × 2
- **可选课程**：权重 × 1

系统会自动避开整数陷阱（50, 100, 150等常见出价），提升中标概率。
